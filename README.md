# Autotrolej Live

Neovisna web aplikacija za praćenje vozila Autotroleja u stvarnom vremenu.

Funkcije:
- Praćenje autobusa uživo
- Prikaz stanica na karti
- Vozni red stanica
- Omiljene linije
- Mobilni i desktop prikaz

Podaci se preuzimaju iz javno dostupnih izvora i API-ja Autotroleja.

Ova aplikacija nije službeni proizvod Autotroleja.
